# Pokdeng Card Game (TypeScript)

A simple Pokdeng card game implemented in TypeScript for the command line.

## Getting Started

### 1. Install dependencies

```npm install```

### 2. Run the game

```npm run start```

### 3. Run tests

```npm test```

## Project Structure

- `src/index.ts` — Main entry point, handles user interaction.
- `src/card.ts` — Card logic and game rules.
- `src/card.test.ts` — Jest tests for game logic.

## Requirements

- Node.js >= 16
- npm
